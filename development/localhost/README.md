## _**How to add a custom domain to localhost**_

- **Windows**
    1. Navigate to this folder

            C:\Windows\System32\drivers\etc

    2. Open this file

            hosts

    3. At the very bottom add a new custom domain
    4. Example

            127.0.0.1 local-planet-link.com
    
    5. Save the file
